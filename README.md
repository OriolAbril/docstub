# docstub

> [!NOTE]
> In early development!

A command line tool to generate Python stub files (PYI) from type descriptions
in NumPyDoc style docstrings.


## Installation

```shell
pip install docstub
```


## Usage & configuration

- Allow extending of the parser (grammar) and the transformer via static files.
